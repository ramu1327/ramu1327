# 💫 About Me:

I help businesses deliver high-quality software by building reliable, scalable, and maintainable automation frameworks.<br><br>My expertise includes designing automation solutions using Selenium, Java, TestNG, Maven, Cucumber BDD, REST Assured, API Testing, SQL, Git, Jenkins, and CI/CD.
